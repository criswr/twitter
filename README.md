# Simulador de Twitter
\- ⚡ Características:
*   \*\*Ingreso con usuario y contraseña\*\*
*   \*\*Escribir twit (240 caracteres)\*\*
*   \*\*Menú para cambiar usuario, escribir nuevo twit, buscar twits o salir\*\*
*   \*\*Búsqueda de twits filtrados por palabra clave\*\*
